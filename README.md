# The HR AI Architect

A framework and resource site helping HR professionals adopt AI safely and effectively,
built from real implementation work rather than theory.

## The Four-Pillar Framework

| Pillar | What it covers |
|--------|----------------|
| **Pains** | The recurring HR problems AI can actually solve (and the ones it can't) |
| **Prompts** | Production-tested prompt patterns for HR workflows |
| **Processes** | How to roll out AI in an HR team: governance, training, adoption |
| **Frameworks** | Decision tools: build vs. buy, risk assessment, vendor evaluation |

## Why This Exists

Most HR AI content is written by people selling something. This is written by a
practitioner. I built it while serving as the HR AI subject matter expert at a global
cybersecurity software company, leading an HR team's full AI adoption: tool selection,
training program design, governance, and live automation builds.
